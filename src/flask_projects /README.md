# Python Practice Files

This repository contains a collection of Python scripts with various programming exercises. Each file demonstrates different concepts and problem-solving techniques.

## `pyhton_files/python_practice_day_1.py`

*   Reads a file named `sample.txt`.
*   Counts the occurrences of each word in the file.
*   Finds the 3 most common words and their counts.
*   Contains a function `count_words` that encapsulates this functionality.

## `pyhton_files/python_practice_day_2.py`

*   Finds the maximum value in a list.
*   Demonstrates list comprehension and string slicing.

## `pyhton_files/python_practice_day_3.py`
